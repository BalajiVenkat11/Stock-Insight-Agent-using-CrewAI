# CrewAI Financial Agents

A multi-agent system using CrewAI and LLMs to fetch stock prices, gather financial news, and generate actionable insights.

## Overview

This project demonstrates a structured, scalable approach:

- **Agents:** Independent units with defined roles and expertise.  
- **Tools:** Encapsulated functions for interacting with APIs or performing computations.  
- **Tasks:** Workflows that orchestrate agents and tools.  
- **Memory & Context:** Agents can retain information across tasks.  
- **LLM Integration:** Guides reasoning and output generation.  

## Libraries Used

- CrewAI  
- LangChain / langchain_openai  
- yfinance  
- DDGS  
- Pydantic  

## Features

- Multi-agent orchestration  
- Modular design for agents, tools, and tasks  
- Handles structured outputs  
- Supports error handling and logging  
- Scalable and maintainable  

## Benefits

- Simplifies large system instructions into modular components  
- Clear separation of responsibilities  
- Supports multi-turn, multi-agent workflows  
- Facilitates testing and debugging
